# Employee Management System (Python OOP Project)

This is a console-based Employee Management System built using Python.

## Features
- Add Employee (Director / Manager / Teacher)
- Search Employee
- Modify Employee
- Delete Employee
- Display All Employees

## Concepts Used
- Object Oriented Programming (OOP)
- Inheritance
- Polymorphism
- List Data Structure
- Console Menu System

## How to Run
python employee_management_system.py
